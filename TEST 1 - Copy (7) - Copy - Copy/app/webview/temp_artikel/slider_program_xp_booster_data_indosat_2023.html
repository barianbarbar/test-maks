Program XP Booster: Indosat Freedom

Perbanyak transaksi paket data Indosat Freedom dan kumpulkan XP lebih banyak untuk naik level lebih cepat!

Melalui XP Booster: Indosat Freedom, kamu bisa dapatkan hingga 750 XP.

Cek dulu syarat dan ketentuannya berikut ini:

  • Program XP Booster: Indosat Freedom berlangsung mulai dari 6 Oktober s.d. 5 November 2023.
  • Program XP Booster: Indosat Freedom membuat semua produk yang berlaku mendapatkan XP lebih banyak dibandingkan XP normal.
  • Program XP Booster: Indosat Freedom hanya berlaku untuk setiap pembelian produk Paket Data Indosat tertentu dengan rincian bonus XP sebagai berikut:
    1. Freedom Internet 9 GB dari 200 XP menjadi 400 XP.
    2. Freedom Internet 16 GB dari 200 XP menjadi 400 XP.
    3. Freedom Internet 70 GB dari 500 XP menjadi 750 XP
  • Mitra akan langsung menerima XP setelah status transaksi dinyatakan sukses di aplikasi.
  • Bonus XP dapat terus diperoleh setiap kali Mitra melakukan transaksi produk yang telah ditentukan selama periode program berlangsung.
  • MuraPay berhak menghentikan program promo sewaktu-waktu.
  • Dengan mengikuti Program ini, Mitra telah membaca, mengetahui, dan menyetujui seluruh syarat dan ketentuan yang berlaku.
